### 기능구현 목록
- [x] 입력값으로 플레이어 생성
    - [x] 같은 이름일 경우 예외 발생
  
- [ ] 입력값으로 배팅금액 생성
    - [ ] 10의 배수가 아닐 경우 예외 발생
    
- [ ] 랜덤으로 카드 뽑기
    - [ ] 같은 카드는 뽑을 수 없음
    - [ ] 한 장 뽑기, 두 장 뽑기
    
- [ ] 이름과 보유한 카드 출력
    - [ ] 딜러의 경우에는 한 장의 카드만 출력
    
- [ ] 카드를 더 뽑을 수 있는지 확인
  
- [ ] 딜러 카드를 한 장 더 뽑아야하는지 확인
  
- [ ] 이름과 보유한 카드, 카드 합계출력 
  
- [ ] 수익 출력
    - [ ] 수익 계산
